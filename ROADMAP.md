# ROADMAP

## Next Layer

### 1. Pattern Engine
Detect:
- HRV down multiple nights
- sleeping HR rising
- respiratory rate elevated
- sleep compression clusters
- deep / REM distortion
- unusual combinations relative to personal baseline

### 2. Memory Engine
Remember:
- what pattern showed up
- what happened after
- whether you felt fine
- whether workload was high
- whether you got sick
- whether one long sleep fixed it
- whether caffeine or stimulation masked it

### 3. Illness Signal Detection
Watch for:
- HRV below normal band
- sleeping HR above normal band
- respiratory rate trending high
- skin temperature deviation shift
- architecture disruption across multiple nights

Output style:
- unusual for you
- monitor recovery
- possible illness-watch pattern
- similar to past fatigue / strain cluster

### 4. Baseline Auto-Learning Window
System should learn:
- 7-day rolling baseline
- 21-day rolling baseline
- personal HRV band
- personal sleeping HR band
- personal respiratory band
- personal sleep architecture tendencies
- rebound timing
- crash timing

### 5. Future AI Inference Layer
Purpose:
- compare today to history
- detect meaningful clusters
- remember outcomes
- generate cautious personal interpretations

Should sound like:
- This pattern is unusual for you.
- Similar clusters previously preceded low-energy days.
- Respiratory rate and sleeping HR are both elevated relative to baseline.
- Monitor for overload, illness, or recovery disruption.

Should NOT sound like:
- You are sick.
- You have a condition.
- Diagnostic certainty language.

## Goal
Move from:
tracker / dashboard

To:
adaptive personal physiological intelligence system
